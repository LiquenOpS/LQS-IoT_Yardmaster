## 🎨 顏色動態變化效果

### frequency_wave (頻率波動)
```
音訊: 低音 ──▶ 中音 ──▶ 高音
顏色: 🔴紅  ──▶ 🟢綠  ──▶ 🔵藍

視覺:
時刻1: 【🔴🔴🔴🔴🟡🟡🟡🟡🟢🟢🟢🟢🔵🔵🔵🔵】
時刻2: 【🟡🟡🟡🟡🟢🟢🟢🟢🟢🟢🟢🟢🔵🔵🔵🔵】(高音增加)
時刻3: 【🔴🔴🔴🔴🔴🔴🟡🟡🟡🟡🟢🟢🟢🟢🔵🔵】(低音增加)

特點: 從中心流動，顏色隨頻率實時變化
```

### color_wave (顏色波動)
```
音訊組成:
- 低音多 ──▶ 整體偏紅/橙
- 中音多 ──▶ 整體偏綠/青
- 高音多 ──▶ 整體偏藍/紫

視覺:
重低音: 【🔴🟠🟡🟠🔴🟠🟡🟠🔴🟠🟡🟠🔴🟠🟡🟠】(波浪狀)
電子樂: 【🔵🟣💜🔵🟣💜🔵🟣💜🔵🟣💜🔵🟣💜🔵】
平衡音: 【🟢🟡🟠🔴🟡🟢🔵💜🟣🔵🟢🟡🟠🔴🟡🟢】

特點: 波浪動畫 + 頻率混合顏色
```

### beat_pulse (節拍脈衝)
```
平常: 【🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢🟢】(暗)
節拍!: 【⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪】(全亮!)
之後: 【🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵🔵】(換色)
節拍!: 【⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪⚪】(再閃!)
之後: 【🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣🟣】(再換色)

特點: 脈動 + 節拍時換色並閃光
```

### waterfall (瀑布)
```
時間 ▼

T1: 【🔴⬛⬛⬛⬛⬛⬛⬛】 ← 高能量低音
T2: 【🟢🔴⬛⬛⬛⬛⬛⬛】 ← 高能量中音出現
T3: 【🔵🟢🔴⬛⬛⬛⬛⬛】 ← 高能量高音出現
T4: 【⬛🔵🟢🔴⬛⬛⬛⬛】 ← 持續流動
T5: 【⬛⬛🔵🟢🔴⬛⬛⬛】
T6: 【⬛⬛⬛🔵🟢🔴⬛⬛】

特點: 頻譜歷史視覺化
```

## ✨ 粒子/動態效果

### blurz (模糊光點)
```
頻段: │ 低 │ 中 │ 高 │
活躍: │ ✓  │ ✗ │ ✓  │

視覺: 【🔴💥⬛⬛🔵💥⬛⬛⬛⬛】
     │   │     │   │
     低   空   高   空
     音   白   音   白

拖尾: 【🟠🔴⬛⬛💜🔵⬛⬛⬛⬛】(淡出中)

特點: 頻段對應位置閃爍 + 拖尾
```

### pixels (像素散射)
```
音量低: 【⬛⬛🟢⬛⬛⬛⬛⬛⬛🔵⬛⬛】(少量)
音量中: 【🔴⬛🟢⬛🟡⬛⬛🔵⬛⬛🟣⬛】(中量)
音量高: 【🔴🟠🟢🟡🟡🔵🔵🔵🟣💜🟣🔴】(大量)

下一刻: 【⬛🟠⬛🟡⬛🔵⬛⬛⬛💜⬛⬛】(隨機變化)

特點: 星空般隨機閃爍
```

### puddles (水坑)
```
安靜:   【⬛⬛⬛⬛⬛⬛⬛⬛⬛⬛⬛⬛】

聲音!:  【⬛⬛🟢🟢🟢⬛⬛⬛⬛⬛⬛⬛】(隨機位置)

淡出:   【⬛⬛🟢🟢🟢⬛⬛⬛🔴🔴⬛⬛】(另一處)

持續:   【⬛⬛🟢🟢🟢⬛⬛⬛🔴🔴🔵🔵】(多處)

特點: 音量觸發 + 彩色光點組
```

### ripple (漣漪)
```
節拍檢測!

T1: 【⬛⬛⬛⬛⬛⬛🔴⬛⬛⬛⬛⬛】(中心起點)
T2: 【⬛⬛⬛⬛⬛🔴⬛🔴⬛⬛⬛⬛】(向外1格)
T3: 【⬛⬛⬛⬛🔴⬛⬛⬛🔴⬛⬛⬛】(向外2格)
T4: 【⬛⬛⬛🔴⬛⬛⬛⬛⬛🔴⬛⬛】(向外3格)
T5: 【⬛⬛🟠⬛⬛⬛⬛⬛⬛⬛🟠⬛】(淡出)

多個波紋: 【🔵🟢⬛🔴⬛⬛⬛⬛🔴⬛🟢🔵】

特點: 同心圓擴散波紋
```

### rainbow_spectrum (彩虹頻譜)
```

特點: 彩虹 + 音量調製
```

## 🎨 顏色變化機制對比

```
📍 固定顏色 + 亮度變化
spectrum_bars: 低=紅, 中=綠, 高=藍 (固定)
vu_meter:      綠→黃→紅 (漸變固定)
fire:          紅+橙 (固定)

🎨 固定彩虹 + 亮度調製
rainbow_spectrum: 彩虹色 + 頻譜調製亮度

🌈 時間驅動顏色變化
puddles:      色相隨時間增加
beat_pulse:   節拍時增加色相

🎵 音訊驅動顏色變化 ⭐
frequency_wave: hue = f(主要頻率)
color_wave:     hue = f(低音,中音,高音比例)
waterfall:      hue = f(最強FFT bin)
blurz:          hue = f(FFT bin 位置)

🎲 隨機顏色
pixels:         從音量歷史隨機選擇
ripple:         節拍時隨機(基於時間)
```
